MyString을 이용하면서
- if(str1.compare(str2) == 0)보단 if(str1 == str2)를 하면 어떨까?
- str1.insert(str1.length(), str2)말고 str1 += str2; 를 하면 어떨까?
- str1[10] = 'c'를 하면 어떨까?

이런 생각을 하게 된다. C언어에선 이런 것이 완전히 불가능하다.
+, = , == , []와 같은 기본 연산자들은 모두 C언어에선 기본적으로 정의되어 있는
데이터 타입(int, float 등)에만 사용 가능한 것들이기 때문이다.

이들을 구조체 변수에 사용하는 것은 불가능하다.

하지만 C++에선 사용자 정의 연산자를 사용할 수 있다.
::(범위지정), .(멤버지정), .*(멤버 포인터로 멤버 지정)을 제외한 모든 연산자를 사용할 수 있다.
- 산술 연산자 : +, -, *
- 축약형 연산자: +=, -=
- 비교 연산자 : >=, ==
- 논리 연산자 : &&, ||
- 멤버 선택 연산자 : -> , * (역참조 연산자 *임)
- 증감 연산자 : ++, --
- [](배열연산자)와 ()까지 (함수 호출 연산자)

이 때 이러한 기본 연산자들을 직접 사용자가 정의하는 것을 오버로딩(overloading)한다고 부른다.
이전에 같은 이름의 함수를 인자만 다르게 정의해 사용하는 것을 '함수를 오버로딩 했다.' 라고 하는 것처럼
기본 연산자를 연러분을 설계한 클래스에 맞게 직접 사용하는 것을 '연산자 오버로딩 했다.'라고 말한다.
